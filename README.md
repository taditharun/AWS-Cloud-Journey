## What I Built
- Custom VPC with CIDR 10.0.0.0/16
- Public subnet: 10.0.1.0/24
- Private subnet: 10.0.2.0/24
- NAT Gateway in public subnet
- VPC Peering between two VPCs

## Key Concepts
- IGW = public internet access
- NAT = private subnet internet access
- Route tables control all traffic flow
- VPC Peering = private VPC communication
